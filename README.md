# YOLOv8 Real-Time Object Detection (Webcam)

## 📌 Introduction
This project uses **YOLOv8 (Ultralytics)** with **OpenCV** to perform **real-time object detection using a webcam**.

The program will:
- Capture video from the webcam
- Detect objects in each frame
- Draw bounding boxes around detected objects
- Display object label and confidence score
- Show real-time FPS (Frames Per Second)

Press **Q** to exit the program.
